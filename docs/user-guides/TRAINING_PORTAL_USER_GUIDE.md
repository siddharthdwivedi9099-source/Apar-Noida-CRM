# Training Portal User Guide

## Purpose

This guide explains how training administrators and customer success managers build and assign training using the Training workspace introduced in Phase 17.

## Accessing the workspace

Open **Training** from the navigation. The module appears for roles with Training access and when the tenant has the Training module enabled.

## Dashboard

The top of the workspace shows program and assignment metrics: total and published programs, total/in-progress/completed assignments, average completion, and average feedback rating.

## Catalog tab — authoring programs

1. Click **Create program** (requires create permission) and set title, category, and level.
2. Select the program to open its detail panel.
3. Use **Add module** to create modules, then **Add lesson** to add lessons to a module.
4. Lessons can be article, video, quiz, or interactive; assets can be linked to lessons (placeholder).

## Assignments tab

- Use **Assign training** to assign a program to a user.
- Select an assignment to see its per-lesson progress; use **Complete** to mark a lesson done.
- The assignment's completion percent and status update automatically as lessons are completed.

## My Training tab

The learner portal shows training assigned to you with assigned, in-progress, and completed counts, plus a recommended-training placeholder.

## Permission behavior

- Users without Training access do not see the module.
- Users without create permission cannot create programs.
- Users without content-management permission cannot add modules, lessons, or assets.
- Users without assign permission cannot assign training.
- Any user with training view can update their own lesson progress and submit feedback.

## Current limits

- Asset upload, role-based learning paths, certifications, and recommended training are placeholders.
- AI placeholder actions are visible but deferred until the AI Gateway phase.
